HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar páginas web. Utiliza una serie de elementos, como etiquetas y atributos, para definir el contenido y la presentación de una página. Los elementos de HTML incluyen encabezados, párrafos, enlaces, imágenes, listas, tablas y formularios, entre otros. Al combinarse con CSS (Cascading Style Sheets) y JavaScript, HTML permite la creación de sitios web interactivos y visualmente atractivos, siendo fundamental en el desarrollo web moderno.
